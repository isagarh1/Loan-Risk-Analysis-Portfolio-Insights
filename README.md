# 📊 Loan Risk Analysis & Portfolio Insights

## 🚀 Overview

Financial institutions face significant risk from loan defaults. This project analyzes loan application data to identify high-risk segments, monitor portfolio performance, and support better lending decisions.

The solution combines **SQL-based analysis and Power BI dashboards** to deliver actionable risk insights.

---

## 🎯 Business Objective

* Identify high-risk loan segments
* Analyze loan portfolio performance
* Support data-driven lending and risk management decisions

---

## 📂 Dataset

* ~39,000+ loan applications
* Key attributes:

  * Loan amount, interest rate
  * Debt-to-Income (DTI) ratio
  * Loan purpose and status
  * Geographic distribution

---

## 🛠 Tools & Technologies

* **SQL** – Data extraction, joins, aggregations
* **Power BI** – Dashboard development and KPI tracking
* **Excel** – Data validation and preprocessing

---

## 🔍 Approach

### 1. Data Preparation

* Cleaned and validated loan datasets
* Structured data for analysis using SQL queries

### 2. Data Analysis

* Calculated KPIs:

  * Total funded amount ($435M)
  * Total received amount ($473M)
* Performed segmentation by:

  * Loan purpose
  * Geography
  * Risk categories

### 3. Dashboard Development

* Built interactive Power BI dashboards to track:

  * Loan performance trends
  * Risk distribution
  * Monthly performance metrics

---

## 📈 Key Insights

* **13.8% of loans classified as high-risk (bad loans)**
* High default rates observed in:

  * Debt consolidation loans
  * Specific high-risk regions (e.g., CA, NY)
* Higher DTI ratios strongly correlate with loan defaults

---

## 💡 Business Impact

* Helps identify high-risk lending segments
* Supports improved credit risk monitoring
* Enables better decision-making for loan approvals
